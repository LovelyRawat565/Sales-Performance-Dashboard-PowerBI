## 📌 Project Overview
This project is an interactive **Sales Performance Dashboard** developed using Power BI. It transforms raw sales data into actionable business insights, allowing executives to track revenue growth, identify top-performing product categories, and analyze performance across different regions.

## 🛠️ Key Features
* **Executive KPIs:** Tracked main business metrics like Total Sales ($541K), Total Quantity (39), and Total Profit ($65K) using clean KPI Cards.
* **Monthly Sales Trend:** Implemented a Line Chart with drill-down capabilities to track monthly and daily revenue shifts.
* **Product Performance:** Built a Horizontal Bar Chart analyzing top sub-categories like Laptops and Phones by total sales.
* **Interactive Slicers:** Added a Region-wise List Slicer (North, South, East, West) to filter the entire dashboard dynamically.

## 🚀 Tech Stack & Skills Used
* **Tool:** Power BI Desktop
* **Data Source:** CSV File (Sales Data)
* **Skills:** Data Loading, UI/UX Dashboard Layout, Dynamic Filtering (Slicers), Chart Types Selection.
